Customer Shopping Behavior Analysis
Overview

This project focuses on analyzing customer shopping behavior using Python, SQL, and Power BI. The analysis helps identify customer purchasing patterns, revenue insights, discount effectiveness, product performance, and customer segmentation.

The project combines:

Python for data cleaning and exploratory data analysis (EDA)
SQL for business queries and analytical insights
Power BI for interactive dashboard visualization
